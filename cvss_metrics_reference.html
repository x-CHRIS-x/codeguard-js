JSentinel — CVSS v3.1 Rule Metrics Reference

All base scores computed using the official CVSS v3.1 formula from the FIRST.org specification document (Section 7.1 & 7.4). Scores verified to match the FIRST.org online calculator. Metrics are pre-assigned per rule using the CVSS v3.1 User Guide's guidance to assume the reasonable worst-case implementation scenario, since JSentinel performs static analysis and cannot evaluate runtime context.

Official CVSS v3.1 Base Score Formula (Section 7.1)

ISS = 1 − (1−C) × (1−I) × (1−A)

If Scope = Unchanged: Impact = 6.42 × ISS
If Scope = Changed: Impact = 7.52 × (ISS−0.029) − 3.25 × (ISS−0.02)¹⁵

Exploitability = 8.22 × AV × AC × PR × UI

If Impact ≤ 0: BaseScore = 0
If Scope = Unchanged: BaseScore = Roundup(min(Impact + Exploitability, 10))
If Scope = Changed: BaseScore = Roundup(min(1.08 × (Impact + Exploitability), 10))

Roundup = ceiling to nearest 0.1

Metric weights (Section 7.4):
AV: N=0.85 A=0.62 L=0.55 P=0.20
AC: L=0.77 H=0.44
PR (Scope=U): N=0.85 L=0.62 H=0.50
PR (Scope=C): N=0.85 L=0.68 H=0.50
UI: N=0.85 R=0.62
C/I/A: N=0.00 L=0.22 H=0.56
